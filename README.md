# email-marketing-automation-n8n
Automated B2B email outreach system built using n8n.
This n8n automation fetch new emails to send from google sheet, and automatically send at schedule time, with follow-ups on same thread.

Features:
• Outreach data storage in Google Sheets (Lead generation and outreach data building is shown in my another project)
• Automated email campaign.
• Automated follow ups
